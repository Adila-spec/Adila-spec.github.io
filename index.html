📐 Key Formulas at a Glance

MetricFormulaPlain English
PrecisionTP / (TP + FP)Of all predicted positives, how many were actually positive?
Recall (Sensitivity)TP / (TP + FN)Of all actual positives, how many did we catch?
F1 Score2 × (P × R) / (P + R)Harmonic mean of precision and recall
Accuracy(TP + TN) / TotalOverall correct predictions
AUC-ROCArea under ROC curveProbability model ranks a positive above a negative; 0.5 = random, 1.0 = perfect
VarianceE[(X − μ)²]Average squared deviation from the mean
BiasE[ŷ] − ySystematic error — how far off your average prediction is from truth

🔁 Basic RAG Pipeline (LangChain)

StepWhat happensLangChain tool
1. LoadIngest documents (PDF, web, CSV)DocumentLoaders
2. SplitChunk text into manageable piecesRecursiveCharacterTextSplitter
3. EmbedConvert chunks to dense vectorsOpenAIEmbeddings / HuggingFaceEmbeddings
4. StoreSave vectors to a vector DBFAISS / Chroma / Pinecone
5. RetrieveSimilarity search for top-k chunksvectorstore.as_retriever()
6. GenerateLLM answers using retrieved contextRetrievalQA / LCEL chain

⚡ Encoder vs Decoder Models — When to Use Which

PropertyEncoder (e.g. BERT)Decoder (e.g. GPT)Encoder-Decoder (e.g. T5)
ArchitectureBidirectional — sees full contextCausal / left-to-right onlyEncoder reads input, decoder generates output
Pretraining taskMasked Language Modeling (MLM)Next-token prediction (CLM)Text-to-Text (seq2seq)
Best forEmbeddings, classification, NER, retrievalText generation, chat, completionTranslation, summarization, Q&A
RAG role✅ Retrieval (embedding the documents)✅ Generation (answering the question)Either, depending on task
ExamplesBERT, RoBERTa, sentence-transformersGPT-4, Llama, Mistral, ClaudeT5, BART, Flan-T5
